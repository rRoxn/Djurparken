För att köra skapa ett virutell miljö sedan:

CLI
Kopiera kod
python -m venv env
source env/bin/activate    # macOS/Linux
env\Scripts\activate       # Windows
Installera beroenden:

CLI

pip install -r requirements.txt
Starta applikationen:

CLI

python src/main.py

Nytt är relativa sökvägar för bilder. 
Tester är ej uppdaterade för detta.
